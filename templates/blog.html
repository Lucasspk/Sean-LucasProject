Sean and Lucas's Senior Project Blog

The main goal of this project is to ultimately create a "how to guide" to computer parts. We wish to learn Python code and put this knowledge toward the creation of a website with information about computer parts. We would like to inform users of how the parts work, how they are made, and why they are priced the way that they are. Ideally users will leave the website informed and ready to purchase the best value computer part for their purpose.
